# Launch Pack: zh-TW

## Positioning

Pie 是一個會動手用瀏覽器的開源 AI 助理。文案以「能做什麼、什麼變輕鬆」為主，避免技術黑話，隱私只用平實說法：你的 key 留在裝置上，無追蹤。

## Chrome Web Store

### Name

Pie · 開源瀏覽器 AI Agent · 多分頁自動化

### Short Description

開源 Chrome AI Agent：讀網頁與 PDF、跨分頁自動化。自帶 API Key（支援 11 家）或訂閱 Pie，無遙測。

### Long Description

Pie 是一個會動手用瀏覽器、而不只是陪你聊天的 AI 助理。它開在 Chrome 側邊欄裡，你工作時一直在那兒：用大白話說出你想做什麼，它就在你眼前的頁面上把事辦好。

• 對你正在看的網頁或 PDF 提問 —— 摘要、重點、直接給答案。
• 把多步任務交給它 —— 比較選項、填表單、跨分頁彙整資訊。
• 整理分頁、連網搜尋、在文件和編輯器裡寫東西、把結果匯出成檔案。
• 常做的任務存成一鍵指令，或設成定時自動跑。

模型隨你接：自帶 API Key（支援 11 家）—— 免費，資料留在你裝置上；或訂閱 Pie、省去一切設定。無論哪種，Pie 都不收集任何追蹤或統計。

開始用：
1. 安裝 Pie。
2. 開啟側邊欄。
3. 新增你的 key，或登入訂閱。
4. 輸入一個任務，讓 Pie 在頁面上動手。
