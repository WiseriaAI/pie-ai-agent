## Chrome Web Store

### Name

Pie · 开源浏览器 AI Agent · 多 Tab 自动化

### Short Description

开源 Chrome AI Agent：读网页与 PDF、跨标签页自动化。自带 API Key（支持 11 家）或订阅 Pie，无遥测。

### Long Description

Pie 是一个会动手用浏览器、而不只是陪你聊天的 AI 助手。它开在 Chrome 侧边栏里，你工作时一直在那儿：用大白话说出你想做什么，它就在你眼前的页面上把事办好。

• 对你正在看的网页或 PDF 提问 —— 总结、要点、直接给答案。
• 把多步任务交给它 —— 比较选项、填表单、跨标签页汇总信息。
• 整理标签页、联网搜索、在文档和编辑器里写东西、把结果导出成文件。
• 常做的任务存成一键命令，或设成定时自动跑。

模型随你接：自带 API Key（支持 11 家）—— 免费，数据留在你设备上；或订阅 Pie、省去一切配置。无论哪种，Pie 都不收集任何追踪或统计。

开始用：
1. 安装 Pie。
2. 打开侧边栏。
3. 添加你的 key，或登录订阅。
4. 输入一个任务，让 Pie 在页面上动手。
