## Chrome Web Store

### Name

Pie · ブラウザ用のオープンAIエージェント

### Short Description

Chrome 用オープンソース AI エージェント。ページや PDF を読み、タブを自動化。自分の API キー（11 社）かサブスクで利用、テレメトリなし。

### Long Description

Pie は、ブラウザを実際に操作してくれる AI アシスタントです。Chrome のサイドパネルに開いたまま、やりたいことを普段の言葉で伝えるだけで、目の前のページで作業を進めます。

・開いているページや PDF について質問 — 要約、要点、答えがその場で。
・複数ステップの作業をおまかせ — 比較、フォーム入力、複数タブからの情報集め。
・タブの整理、Web 検索、ドキュメントやエディタへの入力、結果のファイル書き出し。
・よくやる作業はワンタップのコマンドに、あるいはスケジュールで自動実行。

モデルの使い方は自由です。自分の API キーを使えば（11 プロバイダー対応）無料で、データは端末内に留まります。キー管理が面倒なら Pie のサブスクで即利用。どちらでも、Pie は追跡や分析を一切収集しません。

はじめ方:
1. Pie をインストールします。
2. サイドパネルを開きます。
3. 自分のキーを追加するか、サインインして購読します。
4. タスクを入力し、Pie にページ上で作業させます。
