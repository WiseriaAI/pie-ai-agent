# Launch Pack: ja

## Positioning

Pie は日本のユーザー向けのオープンソースのブラウザエージェントです。既定では BYOK（自分の API キーを使い、リクエストは Pie のバックエンドを経由しません）。キー管理を避けたい人向けに、任意の公式サブスクリプションも用意しています。コピーは率直で製品志向の日本語にし、過度にカジュアルな表現は避けます。

## Chrome Web Store

### Name

Pie · ブラウザ用のオープンAIエージェント

### Short Description

Chrome 用オープンソース AI エージェント。ページや PDF を読み、タブを自動化。自分の API キー（11 社）かサブスクで利用、テレメトリなし。

### Long Description

Pie は Chrome のサイドパネルで動作します。現在開いているページ（PDF を含む）を読み取り、Web サイトを操作し、タブを整理し、Web を検索し、リッチエディタに書き込み、タスクをスケジュール実行できます。

モデルの接続方法は 2 通りです。11 のプロバイダーから自分の API キーを持ち込む方法では、キーはローカルで暗号化され、選んだプロバイダーにのみ送信され、Pie のバックエンドやプロキシは一切介在しません。あるいは任意の公式サブスクリプション（Google でサインイン）では、リクエストは Pie のゲートウェイを経由して転送されますが、保存されることはありません。いずれの場合も、Pie はプロダクト分析用のテレメトリーを収集しません。

はじめ方:
1. Pie をインストールします。
2. サイドパネルを開きます。
3. 自分の API キーを追加するか、公式サブスクリプションにサインインします。
4. タスクを入力し、Pie に現在のページで作業させます。
