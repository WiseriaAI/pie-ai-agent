# v0.11.1 — Centered greeting empty state

The empty side-panel state moves from a top-aligned READY label + skill suggestion cards to a vertically centered random greeting + subtitle. 7 bilingual greeting variants ship — one is picked per side-panel mount, and switching locale re-translates the same line instead of re-rolling.

## What changed

- **New empty state**: READY badge + readyHeadline + skill suggestion cards → centered greeting + single-line subtitle
- **7 bilingual greetings**: warm, friend-tone variants in English and 中文; mount-time random pick
- **Locale-stable**: greeting key is memoized at mount with `useMemo([])`, so locale toggles translate the same line instead of selecting a new one

## Why

- Skill suggestion cards were noise once users knew which skill to invoke
- Centered layout with fewer cards keeps the first frame quieter, fitting Pie's BYOK-tool tone

## Unchanged

- Existing sessions / skills / instances untouched
- No manifest permission changes — silent upgrade
